La plataforma Java

Que es java

Por definición, una plataforma es el ambiente hardware o software sobre el cual se puede ejecutar un programa.

  • La máquina virtual de Java (Java Virtual Machine, JVM)
  • La Interfaz de Programación de Aplicaciones Java (Java Application Programming Interfaces, Java API)
  • La JVM es un intérprete. Los intérpretes son programas que permiten traducir el código creado por un programador en un lenguaje de alto nivel a código máquina.

Ediciones de la Plataforma Java

Se puede distinguir las siguientes cuatro ediciones de Java:

La configuración de una plataforma Java afecta fundamentalmente los servicios que esta ofrece, las herramientas adicionales que incorpora y también aspectos menos visibles, tales como el modo de funcionamiento de la JVM.

  1. Java SE
  2. JavaFX
  3. Java EE
  4. Java ME
Edicion Descripcion
Java SE conocida como Standard Edition es la edición más difundida de la plataforma Java. Incorpora el JDK
Java FX originalmente JavaFX era una alternativa a Java SE
Java EE denominada Enterprise Edition de la plataforma Java

Características de Java

Características principales

Las características principales que nos ofrece Java respecto a cualquier otro lenguaje de programación, son:

  1. Sencillo
  2. Orientado a objetos
  3. Distribuido
Caracteristica Descripcion
Sencillo La sintaxis de Java es similar a la de los lenguajes C y C++
Orientado a objetos Salvo los tipos de datos primitivos, todo en Java es concebido como un objeto.
Distribuido Java implementa los protocolos de red estándar, lo que permite desarrollar aplicaciones cliente/servidor en arquitecturas distribuidas,
